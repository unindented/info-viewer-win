iNFO 1.0.2 released (2005-08-07)

Improved the installation process, so now you can select which components to install (additional language files, shortcuts, etc.). Also added new options to the "Preferences" dialog that let you select the font and colors used by the application.

iNFO 1.0.1 released (2005-07-23)

Small changes, the main one being that the application now stores the width and height of the window on close.

iNFO 1.0.0 released (2005-07-13)

My first public release! Check it out in the downloads section. You've got some nice installers for Windows, and the sources of course.
